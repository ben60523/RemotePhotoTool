RemotePhotoTool

RemotePhotoTool is a remote image capture application for Canon based cameras. It supports taking HDR and Panorama photos.

Download software »

Photo Modes

Different Photo Modes allow to streamline the process to take photos:

  • Normal shooting
  • HDR processing
  • Panorama pictures

External Tool integration

RemotePhotoTool integrates external tools to further process the images taken. Supported tools are PhotoMatix for processing HDR images, and Hugin for stitching Panorama images together.

Lua scripting

RemotePhotoTool allows you to run scripts to exactly control your camera (while connected). The scripts are written in the popular scripting language Lua.
